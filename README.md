# DUA - Dungeons &amp; Dragons Unlimited Adventure

## Commands
* `r[oll] [<n>]d<s>[+|-<m>]`: Roll _n_ _s_-side dices and apply the _m_ modifier.
Example: `roll 2d6+1`, `r d6`
